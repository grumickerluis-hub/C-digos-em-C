# 💻 Exercícios em Linguagem C

Este repositório reúne exercícios desenvolvidos durante meus estudos de programação em linguagem C ao longo da graduação em Ciência da Computação.

## Objetivo

Praticar e consolidar os principais conceitos da programação estruturada, desenvolvendo soluções para problemas propostos em aulas, listas de exercícios e plataformas de programação.

## Conteúdos praticados

- Entrada e saída de dados
- Variáveis e operadores
- Estruturas condicionais (`if` e `switch`)
- Estruturas de repetição (`for`, `while` e `do-while`)
- Funções
- Vetores
- Matrizes
- Strings
- Ponteiros
- Modularização
- Manipulação de arquivos
- Alocação dinâmica de memória

## Exercícios

Este repositório contém diversos exercícios, incluindo:

- Sequência de Fibonacci
- Operações com vetores
- Manipulação de matrizes
- Cálculo de médias
- Busca e ordenação
- Exercícios de lógica
- Problemas resolvidos do Beecrowd
- Exercícios propostos durante a graduação

## Tecnologias utilizadas

- Linguagem C
- Visual Studio Code
- GCC (GNU Compiler Collection)

## Objetivo do repositório

Registrar minha evolução na linguagem C, documentando os exercícios desenvolvidos e os conceitos aprendidos ao longo da graduação.
